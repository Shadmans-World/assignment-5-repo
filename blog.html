Important FAQ

What is the Document Option Model (DOM)?

DOM is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from dom?

You can select an element from the DOM using JavaScript methods like document.getElementById(), document.querySelector(), document.getElementsByClassName(), or document.getElementsByTagName() to access specific elements.

What is event delegation in the context of the DOM and why it is useful?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element to manage events for its child elements. Instead of adding event listeners to individual child elements, the parent handles the events by capturing them as they bubble up through the DOM (event bubbling).

Why it's useful:

Performance: Reduces the number of event listeners needed, saving memory and improving efficiency, especially with dynamic elements.
Dynamic Elements: Works well when child elements are dynamically added, as the parent listener will still capture their events. Simplifies Code: Reduces repetitive event binding logic, making the code cleaner and more maintainable.
For example, handling clicks on multiple buttons inside a container without adding listeners to each button individually is a common use case.

How Do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles in the DOM using the following methods:


1. Manipulating Attributes:

setAttribute(): Set or update an attribute.
Example: element.setAttribute('src', 'newImage.jpg');

getAttribute(): Retrieve the value of an attribute.
Example: let value = element.getAttribute('src');

removeAttribute(): Remove an attribute.
Example: element.removeAttribute('disabled');
